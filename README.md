## 其它
使用技术：
- eggjs
- rss-parser
- handlebars

### 工作流程
配置RSS源数组，每天按照配置的RSS源数组逐个爬取数据，rss-parser解析数据构造JSON数组然后使用handlebars按照编写好的模版生成md文件。
通过脚本自动提交更新到git上。
